
---

1. **Update the package index:**
    
    ```bash
    sudo apt update
    ```
    
2. **Install Maven:**
    
    ```bash
    sudo apt install maven -y
    ```
    
3. **Verify the installation:**
    
    ```bash
    mvn -version
    ```
    
    This command will display the installed Maven version, confirming that it is correctly installed.
