Variable SWT fails when 
- Negative numbers present(we cannot adjust window , because future value can be -v)
- Exact match needed (== k) (count subarray of sum k) 

Variable Sliding Window Works When:
- All numbers are **positive**"
- Condition is **<font color="#ffc000">range-based</font>** ( < k, <= k )
